\doxysubsection{Подробное описание}
Класс для перемещения аватара игрока полётом 

\begin{DoxyAuthor}{Авторы}
Пивко Артём 
\end{DoxyAuthor}
\begin{DoxyVersion}{Версия}
1.\+0 
\end{DoxyVersion}
\begin{DoxyDate}{Дата}
13.\+05.\+20
\end{DoxyDate}
Перемещение осуществляется нажатием на заданную кнопку. При её нажатии аватар игрока начинает перемещаться по направлению от шлема к руке, на которой нажата кнопка. Чем дальше рука от шлема, тем быстрее летит аватар.
