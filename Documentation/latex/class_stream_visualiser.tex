\doxysubsection{Подробное описание}
Класс для контроля потоковых визуализаторов 

\begin{DoxyAuthor}{Авторы}
Пивко Артём 
\end{DoxyAuthor}
\begin{DoxyVersion}{Версия}
1.\+0 
\end{DoxyVersion}
\begin{DoxyDate}{Дата}
14.\+05.\+20 
\end{DoxyDate}
\begin{DoxyWarning}{Предупреждения}
Функционирует постоянно. Не только во время работы программы, но и в редакторе.
\end{DoxyWarning}
Этот класс визуализатор использует систему частиц для визуализации \char`\"{}потоковых\char`\"{} данных, таких как параметры электропроводки, водяного или воздушного охлаждения. Система частиц должна иметь включенными модули Emission, Shape, Trails и Renderer. Рекомеднуется использовать только как часть префаба Steam\+Visualiser
