Класс для контроля интерфейса монитора оборудования

Этот класс занимается контролем текстового интерфейса, отображающего все данные, имеющиеся о единице оборудования. ~\newline
Для включения/выключения интерфейса используйте метод Turn\+TV через кнопки на оборудовании.
