Синглтон-\/конвертер входных данных в настройки визуализаторов

\begin{DoxyWarning}{Предупреждения}
Этот скрипт должен быть размещен на сцене в ЕДИНСТВЕННОМ экземпляре. Если его не будет, не будут работать никакие визуализаторы. Если их будет много, останется только один. ~\newline
 Для работы скрипта требуется заполнение в инспекторе настроек преобразовнаия цветов.
\end{DoxyWarning}
Этот класс занимается превращением цифровых данных с источников (температуры, влажности и т. д.) в параметры для визуализатров в Unity (Материалы различных цветов, ... ).

\begin{DoxyReturn}{Возвращает}
Конвертированный в соответствии с температурой цвет Рекомендуется использовать вместо этого метода Temp\+To\+Color~\newline
 Метод использует значения настроек цветов из этого файла как границы перехода цветов\+: ~\newline
 • Если температура $<$= abs\+Cold, то цвет будет чистым синим с обычной прозрачностью ~\newline
 • Если abs\+Cold $<$= температура $<$ allowed\+Start, то цвет будет голубым и приблежаться к бирюзовому с ростом температуры. Прозрачность нормальная ~\newline
 • Если allowed\+Start $<$= температура $<$ optimal\+Start, то цвет будет бирюзовым и приблежаться к зеленому с ростом температуры. Прозрачность тоже будет расти до полного исчезновения ~\newline
 • Если optimal\+Start $<$= температура $<$ optimal\+Finish, Объект полностью прозрачен~\newline
 • Если optimal\+Finish $<$= температура $<$ allowed\+Finish, то цвет будет желтым и приблежаться к оранжевому с ростом температуры. Прозрачность тоже будет падать до нормального значения ~\newline
 • Если allowed\+Finish $<$= температура $<$ abs\+Heat, то цвет будет оранжевым и приблежаться к красному с ростом температуры. Прозрачность нормальная ~\newline
 • Если температура $>$ abs\+Cold, то цвет будет чистым красным с обычной прозрачностью 
\end{DoxyReturn}
