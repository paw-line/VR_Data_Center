1) Преобразование имени датчика в имя типа ~\newline

Если она присутствует в полученном в метод типе, возвращается значение этой пары. ~\newline
Например, если как ключ поставить \char`\"{}?\+Python\+Room\+T\char`\"{}, а как значение roomtemp, то при принятии аргумента \char`\"{}\+Python\+Room\+T5\char`\"{} будет возвращено roomtemp.
