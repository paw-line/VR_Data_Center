Подробное описание

Класс для преобразования типов, получения единиц измерения и переводов.

Предупреждения
Этот скрипт должен быть размещен на сцене в ЕДИНСТВЕННОМ экземпляре. Если его не будет, не будут работать никакие визуализаторы. Если их будет много, останется только один.
Для корректной работы компонента требутеся заполнить словари в редакторе. 4
Необходимо сделать:
Сделать TranstypeToRussian

Этот класс-синглтон используется как глобальный сервис, выполняющий 3 функции:
1) Преобразование имени датчика в имя типа
2) Преобразование имени типа в символ, отражающий единицу измерения (например, символ процента "%")
3) Перевод имени датчика в понятное название

◆ TransGeneralTypeToUnit()

string UniversalTranslator.TransGeneralTypeToUnit ( string  type)

Метод преобразования типа в единицу измерения

Аргументы
typeТип данных
Возвращает
символ(ы) единицы измерения в формате string Ищет в словаре generalTypeToUnitDictionary ключ, полученный в аргументе type и возвращает его значение.
В случае, если ключ не был найден, выводит ошибку в консоль и возвращает пустую строку, что позволяет оторбражать величину просто как числовое значение.

◆ TransTypeToGeneralType()

string UniversalTranslator.TransTypeToGeneralType ( string  type)

Метод преобразования имени датчика в тип

Аргументы
typeИмя датчика
Возвращает
Общий тип данных датчика string Ищет в словаре typeToGeneraTypeDictionary ключ, полученный в аргументе type и возвращает его значение.
При заполнении словаря можно использовать специальный синтаксис: если ключ словаря начинается с символа вопросительного знака "?", всё что находится после него испольщуется не как ключ, а как строка для поиска. Если она присутствует в полученном в метод типе, возвращается значение этой пары.
Например, если как ключ поставить "?PythonRoomT", а как значение roomtemp, то при принятии аргумента "PythonRoomT5" будет возвращено roomtemp.
