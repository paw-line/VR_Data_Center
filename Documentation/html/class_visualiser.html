Универсальный метод снятия показаний с визуализатора

Аргументы
visTypeТип визуализатора
dataTypeТип данных
topicMQTT-топик данных
Возвращает
Визуализируемые данные в формате float

Унаследованные реализации функции обязаны будут возвращать 4 параметра: опорные данные визуализатора, тип визуализатора, тип данных, MQTT-топик данных.
Функция необходима для работы сканера визуализаторов. Внимание, передаваемые в метод параметры при его вызове должны стоять после ключегого слова out.

Замещается в NodeVisualisator, StreamVisualiser, ColorChanger, BarVisualiser и Callout.
