Подробное описание

Класс для контроля интерфейса монитора оборудования

Авторы
Пивко Артём
Версия
1.0
Дата
14.05.20

Этот класс занимается контролем текстового интерфейса, отображающего все данные, имеющиеся о единице оборудования.
Для включения/выключения интерфейса используйте метод TurnTV через кнопки на оборудовании.
