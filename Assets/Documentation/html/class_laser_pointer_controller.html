Подробное описание

Класс для контроля лазерного указателя, сканера и меню.

Этот класс занимается контролем инструментов взаимодействия пользователя с моделью: лазерного указателя для управления меню, сканера показаний визуализаторов и наручного меню. Для корректной работы скрипт требуется разместить на том же объекте, где находится скрипт руки виртуального аватара.

Метод сканирования визуализаторов Метод производит проекцию луча вдоль отображаемого SteamVR_LaserPointer луча, вычленяет из объекта, пересеченного лучом компонент, наследованный от Visualiser. После чего производится вызов метода Scan и вывод полученных данных на интерфейс сканера.
