Этот класс занимается контролем высоты и цвета столбца-\/визуализатора, которым должен являться объект, к которому прикремлен скрипт. Контроль происходит в соотвествии с данными из источника данных \mbox{\hyperlink{class_data_source}{Data\+Source}} source и внутренних настроек объекта.
