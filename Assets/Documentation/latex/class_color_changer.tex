Что у нас есть\+:~\newline
 • Целевой R\+GB цвет ~\newline
 • Время смены ~\newline
 • Изначальный R\+GB цвет ~\newline
 • Delta\+Time – время смены кадра в Unity ~\newline
 R\+GB цвет состоит из трёх компонент, собственно R G и B (ещё есть альфа-\/канал, но в данном случае он не важен). ~\newline
 Для плавной смены цвета нужно чтобы каждый кадр значение цвета менялось в сторону целевого цвета на величину пропорциональную Delta\+Time. Должна соблюдаться пропорция\+: ~\newline
 Delta\+Color/(ЦелевойЦвет-\/ИзначальныйЦвет) = Delta\+Time / ВремяСмены ~\newline
 Где Delta\+Color – изменение цвета за кадр ~\newline
 Таким образом ~\newline
 Delta\+Color = Delta\+Time $\ast$ (ЦелевойЦвет-\/ИзначальныйЦвет) / ВремяСмены ~\newline
 При линейной плавной смене цвета таких дельты будет 3\+: для каждого из каналов R\+GB. Если же смена нелинейная, а идет как, например, в нашем случае через зеленый, то дельтировать нужно контролирующий параметр, то есть температуру. \mbox{\Hypertarget{class_color_changer_a833ec8538965116fd92bec4d9213cfef}\label{class_color_changer_a833ec8538965116fd92bec4d9213cfef}}

Конвертированный в соответствии с температурой цвет Метод доступен извне класса, так как им так же пользуется для изменения цветов Node\+Visualiser. ~\newline
На самом деле это костыль и надо сделать универсальный конвертер отдельным классом. ~\newline
 ~\newline
Метод использует значения настроек цветов из этого файла как границы перехода цветов\+: ~\newline
 • Если температура $<$= abs\+Cold, то цвет будет чистым синим с обычной прозрачностью ~\newline
 • Если abs\+Cold $<$= температура $<$ allowed\+Start, то цвет будет голубым и приблежаться к бирюзовому с ростом температуры. Прозрачность нормальная ~\newline
 • Если allowed\+Start $<$= температура $<$ optimal\+Start, то цвет будет бирюзовым и приблежаться к зеленому с ростом температуры. Прозрачность тоже будет расти до полного исчезновения ~\newline
 • Если optimal\+Start $<$= температура $<$ optimal\+Finish, Объект полностью прозрачен~\newline
 • Если optimal\+Finish $<$= температура $<$ allowed\+Finish, то цвет будет желтым и приблежаться к оранжевому с ростом температуры. Прозрачность тоже будет падать до нормального значения ~\newline
 • Если allowed\+Finish $<$= температура $<$ abs\+Heat, то цвет будет оранжевым и приблежаться к красному с ростом температуры. Прозрачность нормальная ~\newline
 • Если температура $>$ abs\+Cold, то цвет будет чистым красным с обычной прозрачностью ~\newline
